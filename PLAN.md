# Homepage Architecture Plan

## Overview
Build a professional resume/portfolio homepage for John Doe, a freelance web developer. The page features a fixed navigation header with site branding and menu links, followed by a full-height hero section that introduces his services with a strong call-to-action.

## Component Structure
- **Header** - Fixed navigation bar with logo/name and menu links (Home, About, Portfolio, Contact)
- **Hero** - Full-screen hero section with headline, subheading, and CTA button(s)
- **Home** - Root page component that composes Header and Hero

## Design Approach
- Clean, modern, professional design suitable for a freelance developer's portfolio
- Responsive layout that works on mobile and desktop
- Clear visual hierarchy with strong typography
- Navigation remains accessible and functional across viewport sizes
- Hero section uses contrasting colors and spacing to draw attention

## Tech Stack
- React with functional components
- CSS Modules or Tailwind CSS for styling (can be configured)
- Simple, no external dependencies initially
