# Resume Website - Implementation Plan

## Overview
Build a React resume website for John Doe, a freelance web developer. The homepage features a hero section with name, tagline, and CTA button, plus a main navigation menu with links to About, Projects, and Contact sections.

## Architecture

### Component Structure
- **Header** — Navigation menu component with responsive links to About, Projects, Contact
- **Hero** — Hero section displaying name, tagline, and CTA button
- **App** — Main page component that renders Header and Hero

### Tech Stack
- React 18+
- Tailwind CSS for styling
- React Router (for navigation links)

### Design Approach
- Mobile-first responsive design using Tailwind CSS
- Clean, professional styling suited for a developer portfolio
- Semantic HTML structure for accessibility
- Navigation links are client-side routing-ready

## Build Order
1. Set up project with Tailwind CSS
2. Build Header component (navigation)
3. Build Hero component (name, tagline, CTA)
4. Integrate in App and verify layout
